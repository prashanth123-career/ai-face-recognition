This is an AI-powered face recognition system for real-time face detection and recognition.

## 🚀 How to Use  
1️⃣ Install required dependencies using `pip install -r requirements.txt`  
2️⃣ Train the model using `python train_face_recognition.py`  
3️⃣ Run real-time face recognition with `python real_time_face_recognition.py`  

## 🛠 Troubleshooting  
If you face issues, ensure that:  
- Python & dependencies are installed  
- The dataset is placed correctly in `data_face_recognition/`
- The model `face_recognition_model.pth` exists in `models/`
